# E-Commerce Web (Vite + React)

This is a simple e-commerce website assignment built with:

- React (Vite)
- React Router
- Axios for API calls
- Tailwind CSS for styling
- Fake Store API (https://fakestoreapi.com/)

## Quick start

1. Install dependencies:
   ```
   npm install
   ```

2. Initialize Tailwind (already configured) and run dev server:
   ```
   npm run dev
   ```

Open http://localhost:5173

The project fetches products from `https://fakestoreapi.com/products` and supports search, category filters and product details pages.
